# 文件加密工具 WebUI

这是一个现代化、响应式的Web管理界面，用于控制文件加密工具。

## 功能特性

- 🎨 精美现代化界面设计
- 📱 完全响应式，适配手机、平板和电脑
- 🔒 实时进度跟踪
- ⚙️ 配置管理
- 📁 文件映射表查看
- ▶️ 加密/解密操作控制
- 📜 实时日志查看

## 使用方法

1. 启动WebUI服务：
   ```
   ./encrypt-tool -webui
   ```

2. 在浏览器中访问：http://localhost:9394

## 功能说明

### 状态面板
- 显示应用当前运行状态
- 显示正在进行的操作模式（加密/解密）
- 实时进度条显示处理进度

### 配置管理
- 设置加密密码
- 选择加密算法（AES/Blowfish/XOR）
- 配置目标路径
- 设置映射文件相关参数

### 文件操作
- 查看已加密文件列表
- 启动加密操作
- 启动解密操作
- 停止当前操作

### 日志面板
- 实时显示操作日志
- 支持日志清空和下载

## 技术实现

- 前端：HTML5 + CSS3 + JavaScript
- 响应式设计：Flexbox + Grid
- 后端：Go语言
- 通信：RESTful API + Server-Sent Events

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 注意事项

- 请确保配置文件中的密码足够复杂以保证安全性
- 加密/解密操作可能需要较长时间，请耐心等待
- 操作过程中请勿关闭浏览器或刷新页面
